---------------------
SVN Command Line Help
---------------------

    svn --help


Repository Check Out:

    svn checkout svn+ssh://inb461@kdcinfo/opt/repository/unix/repos-software repos-software


    svn co -r {2014-10-18} svn+ssh://inb461@kdcinfo/opt/repository/unix/repos-software repos-software    # for a particular date


Repository export (publish)

    svn export   svn+ssh://inb461@kdcinfo/opt/repository/unix/repos-software repos-software


Repository Commit:

    svn commit -m "Log comment goes here"  


Repository Update:

    svn update  


Repository Status:

    svn    status
    svn -v status
    svn    status --no-ignore
    svn    info


Repository List:

     svn list -R -v svn+ssh://inb461@kdcinfo/opt/repository/unix/repos-software

Repository Log:

     svn log -r {2014-09-04}:{2014-09-14}  svn+ssh://inb461@kdcinfo/opt/repository/unix/repos-software


-----
NOTES
-----

You do not need to check out the entire repository. Instead, you can check out the directory 
    that you are working on.

    svn checkout svn+ssh://inb461@kdcinfo/opt/repository/unix/repos-software/Linux/qa-linux      # checks out qa-linux


Don't forget the unix "tree" command to get a good listing of the directory structure
